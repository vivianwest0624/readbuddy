隱私權政策

最後更新：2025-09-07

ReadBuddy（以下稱「本服務」）是一個在瀏覽器端執行的文字轉語音與生字練習工具。為了讓您了解本服務如何處理資料，特此說明如下。

一、我們處理哪些資料

  • 本機儲存（localStorage）：您的文章、單字表、語速/語音設定、（可選）AI 供應商與模型等，僅保留在您的瀏覽器裝置，不會上傳至伺服器。
  • 登入 Google 後（選用）：若您同意連結 Google 帳號，本服務將要求 openid email profile 與 https://www.googleapis.com/auth/drive.appdata 權限，用於在您的 Google Drive appDataFolder 內讀寫單一檔案 readbuddy_articles.json（包含文章與單字）。
  • 第三方字典/翻譯：為產生單字解釋，本服務可能在前端直接呼叫 dictionaryapi.dev 與 MyMemory 翻譯 API；若您輸入 OpenAI API Key，則由您的瀏覽器直接連線至 OpenAI API。

二、我們如何使用資料

  • 在本機：用於顯示、朗讀與管理您的內容，提升使用體驗。
  • 在 Google Drive（選用）：僅於您主動點擊「上傳/下載」時進行雲端同步。
  • 不做廣告追蹤、不出售個資、無伺服器端資料庫。

三、資料保存與刪除

  • 本機資料：您可在瀏覽器清除網站資料（localStorage）以刪除。
  • Google 權限：可於 Google 帳戶 → 安全性 → 第三方存取權 撤銷 ReadBuddy 的權限。
  • Drive 檔案：appDataFolder 為隱藏空間；您可透過本服務未來提供的刪除功能或開發者協助刪除 readbuddy_articles.json。

四、第三方服務

  • Google Identity Services（OAuth 2.0）、Google Drive API
  • dictionaryapi.dev（英文字典）
  • MyMemory 翻譯（英→繁中）
  • （可選）OpenAI API（若您自行提供 API Key）

五、聯絡方式

若您對本政策有疑問，請聯絡開發者：vivianwest@gmail.com。
